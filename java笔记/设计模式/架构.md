# 问题



1. 软件系统开发中常用的软件架构风格有哪些？



- 数据流风格
- 调用/返回风格
- 独立构件风格
- 虚拟机风格
- 仓库风格
- 分布式处理架构风格
- REST混合架构风格







1. 选取一种架构，设计一个软件系统， 并分析采用该架构风格设计的原因。



重要元素

1. 高并发
2. 高可用
3. 高性能
4. 可扩展
